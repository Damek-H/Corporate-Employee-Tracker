# Corporate Employee Tracker

## Description
Easy to use employee tracker that allows the users to track and/or create department, role and employee tables that can store information for each. 

## Features
* Node.js
* Inquirer.js
* MySQL2
* ASCII-art Logo
* Console.table
* JavaScript
